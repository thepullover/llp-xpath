
### Особенности реализации

Операции над элементами:

* \+ добавление элемента
* – удаление элемента
* ? поиск по элементам
*	= изменение имеющегося элемента

Предикаты операторов:

* @ - true
* !@ - false 
* < меньше
* \> больше 
* = равно
* <field>:<value> поиск с подстрокой
